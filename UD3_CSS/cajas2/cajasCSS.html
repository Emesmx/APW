Ejercicio Cajas CSS3

Este es un párrafo de ejemplo con la clase "caja" aplicada. El texto tiene varias líneas para demostrar cómo funciona el relleno, los márgenes y los bordes multicolores. Los bordes superior rojo (4px), derecho azul (6px) e inferior verde (2px) se ven claramente, junto con el fondo amarillo y los márgenes especificados. ¡Perfecto para practicar CSS de cajas!
