Qué es, características y cómo funciona

El HTML5 es un estándar que sirve para definir la estructura, el diseño y el contenido de una página web.
Se trata de un código, un lenguaje (HTML) que define los contenidos que forman parte de una página web, como imágenes, texto, vídeos, juegos, marcos, estilo de letra, etc.
